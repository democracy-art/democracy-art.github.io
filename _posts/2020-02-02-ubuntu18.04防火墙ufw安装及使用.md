# 1. 安装
```
sudo apt-get install ufw
```
# 2. 使用

### 2.1 启动和关闭
```
sudo ufw enable
sudo ufw disable
```
### 2.2 查看工作情况和端口打开信息
```
sudo ufw status verbose
```
### 2.3 添加开放端口
添加一个端口:
```
sudo ufw allow 80/tcp
sudo ufw allow 80/udp
```
添加多个连续端口,用 `:` 分隔
```
allow 11200:11299/tcp
```
### 2.4 添加拒绝端口
同时拒绝 TCP 和 UDP
```
sudo ufw deny 53
```
只拒绝 TCP
```
sudo ufw deny 53/tcp
```
### 2.5 删除存在的规则
```
sudo ufw delete deny 80/tcp
```
```
sudo ufw delete allow 53/udp
```
