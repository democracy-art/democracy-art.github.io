# 1. 材料
- 1.1 鸡翅
- 1.2 葱
- 1.3 姜
- 1.4 蒜
- 1.5 豆豉

# 2. 调料
- 2.1 郫县豆瓣酱
- 2.2 料酒
- 2.3 酱油
- 2.4 糖
- 2.5 淀粉
- 2.6 老抽
- 2.7 香油

# 3. 准备
- 3.1 鸡翅两面切两刀,放盐,放点料酒搅拌腌制
- 3.2 葱姜蒜切碎,豆豉洗一下再切碎

# 4. 步骤
- 4.1 放油炒,葱姜蒜+豆豉+郫县豆瓣酱+料酒+酱油+糖+水
- 4.2 鸡腿+淀粉拌一下+炒好的汁+老抽搅拌+香油,腌制15分钟
- 4.3 蒸 **60** 分钟(**40**分钟，鸡翅硬一点)
