# 1. 材料
- 1.1 西红柿
- 1.2 葱
- 1.3 鸡蛋

# 2. 调料
- 2.1 
# 3. 准备
- 3.1 烧水烫一下西红柿方便去皮
- 3.2 切葱花
- 3.3 西红柿的皮去掉，把里面的水挤掉一些
- 3.4 鸡蛋碗里搅拌,**不** 要放盐

# 4. 步骤
- 4.1 锅里放点油，倒入鸡蛋，炒到金黄，倒出备用
- 4.2 再放入油，西红柿+葱花+酱油+老抽(添色)+糖+盐
- 4.3 再放鸡蛋搅拌,让鸡蛋充分吸收西红柿的汁
