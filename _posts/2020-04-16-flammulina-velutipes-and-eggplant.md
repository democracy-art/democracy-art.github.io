# 1. 材料
- 1.1 金针菇
- 1.2 葱,姜,蒜
- 1.3 茄子

# 2. 调料
- 2.1 柱候酱
- 2.2 黄酒
- 2.3 酱油
- 2.4 蚝油
- 2.5 糖
- 2.6 老抽
- 2.7 淀粉

# 3. 准备
- 3.1 金针菇洗净，切去泥那一段
- 3.2 茄子切条带皮(营养都在皮里面)
- 3.3 葱切小段,姜切片,蒜切片
- 3.4 淀粉放水勾芡

# 4. 步骤
- 4.1 金针菇煮水烫一下,同时放点油(让它保持颜色)
- 4.2 锅放油炸茄子条,炸好出锅备用
- 4.3 锅放油+葱+姜+蒜+柱候酱,炒,+炸好的茄子条+清水+金针菇+黄酒+酱油+蚝油+糖+老抽+生粉勾芡
- 4.4 砂锅放点油润锅，从炒锅转砂锅炖,上菜
