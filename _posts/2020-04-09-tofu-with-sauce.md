# 1. 材料
- 1.1 豆腐(切片)
- 1.2 小米辣椒(切圈)
- 1.3 葱(切圈)
- 1.4 姜(切丝)
- 1.5 鸡蛋

# 2. 调料
- 2.1 香油
- 2.2 淀粉
- 2.3 蚝油 
- 2.4 生抽
- 2.5 白糖

# 3. 步骤
- 3.1 豆腐码盘,双面撒盐,双面撒淀粉
- 3.2 打鸡蛋,用豆腐蘸蛋液
- 3.3 放油煎豆腐,煎到两边金黄盛出备用
- 3.4 姜丝+葱花+小辣椒,炒,放生抽+蚝油
- 3.5 加水+白糖+豆腐继续炒, 最后加点香油,炒香出锅
