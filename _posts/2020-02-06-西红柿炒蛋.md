# 材料:
```
1.西红柿
2.鸡蛋
3.油
4.盐
5.醋
6.白砂糖
7.番茄酱(可选)
8.水淀粉(可选)
```

# 步骤:

1.炒蛋<br>
1.1 鸡蛋打入碗,加入盐,醋(去腥)搅拌<br>
1.2 放油入锅,倒鸡蛋入锅,小火煎蛋一分钟,将蛋煎至定型,出锅倒入碟子,备用<br><br>

2.炒西红柿<br>
2.1 放油入锅,放西红柿,炒均后,放清水,小火烧1分钟<br>
2.2 1分钟后,调味:放盐,放糖(平衡西红柿的酸味)<br>
2.3 这步可选:加番茄酱增加西红柿本味, 加水淀粉将汤汁勾薄芡<br>

3.西红柿鸡蛋一起炒,放点葱,然后出锅
